## basic usage
gdb hello
`l` or `list`    list source code
`b` or `break`   set breakpint
`i`              information
`i b`            view information about breakpoints
`r` or `run` 
`p`           print
`p str`       print str's value
`n`           next
`c` or `continue`
`i locals` view local variables
`quit`
